# 内存分配策略和分配算法

在为进程分配内存时，将涉及到三个问题：

1. 最小物理块数问题

	- **能保证进程正常运行所需的最小物理块数**

3. 物理块的分配策略

   - 固定分配局部置换

   - 可变分配全部置换

   - 可变分配局部置换

4. 物理块分配算法

   - 平均分配算法

   - 按比例分配算法
